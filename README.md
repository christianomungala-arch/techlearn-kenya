 TechLearn Kenya 

A fully responsive multi-page web application built as a front-end development project.
TechLearn Kenya is a fictional online learning platform offering affordable tech courses
to Kenyan learners.


 Live Site

[View Live Site](https://YOUR-USERNAME.github.io/techlearn-kenya)



 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero banner, stats, featured courses |
| About | `about.html` | Mission, values, and team profiles |
| Courses | `courses.html` | Full course catalog with JS filter |
| Blog | `blog.html` | Articles and tech news |
| Gallery | `gallery.html` | Photo gallery of events |
| FAQ | `faq.html` | Frequently asked questions accordion |
| Testimonials | `testimonials.html` | Student reviews and ratings |
| Contact | `contact.html` | Contact form with JS validation |

---

 Technologies Used

HTML5 — Semantic page structure
CSS3 — Custom styles, transitions, dark mode variables
JavaScript (ES6+) — Dark mode toggle, form validation, course filter
Bootstrap 5 — Responsive grid, navbar, cards, modals, accordion
Bootstrap Icons — Icon library throughout the site
Git & GitHub — Version control with incremental commits



 Features

 Fully responsive on desktop, tablet, and mobile
 Consistent navbar and footer across all 8 pages
 Dark / Light mode toggle (preference saved in localStorage)
 Contact form with JavaScript validation
 Filterable course catalog by category
  Bootstrap accordion for FAQ
  Bootstrap navbar that collapses on mobile
